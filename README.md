The script
sets up the following, configured for this specific palette:
1. Niri WM (beta)
 - HTML page on Super+F1
 - Configured layer and window rules
2. Zed, Firefox, GIMP, gucharmap, Dolphin, octoXBPS
3. Kitty
 - With a color scheme sourced and configured from the ![repository](https://github.com/ttys3/oh-my-kitty)
 - Custom ![Qt theme](https://github.com/renamon2/kvantum-rena)
 - Matching ![pine-rose-gtk](https://github.com/rose-pine/gtk) (beta)
4. Rofi
 - Papirus icons
 - drun, filebrowser, run
5. Waybar
 - Tray, window, date & time, pavucontrol, CPU..., Bluetooth, Wi-Fi, power (beta)
